# Avatares de Guías Celestiales

Coloca aquí las imágenes de avatar de cada guía (formato cuadrado 1:1).

## Nomenclatura

- **Avatar cuadrado (obligatorio):** `{guideId}.png`  
  Ejemplos: `aethel.png`, `crono-velo.png`, `luna-vacia.png`, `helioforja.png`, etc.

- **Avatar vertical (opcional):** `{guideId}_vertical.png`  
  Para pantallas de bienvenida o detalle.

## IDs de guías (21)

aethel, crono-velo, luna-vacia, helioforja, leona-nova, chispa-azul, gloria-sincro, pacha-nexo, gea-metrica, selene-fase, viento-estacion, atlas-orbital, erebo-logica, anima-suave, morfeo-astral, shiva-fluido, loki-error, eris-nucleo, anubis-vinculo, zenit-cero, oceano-bit

## Recursos existentes en docs

Puedes copiar o referenciar desde `docs/personajes-misticos/`:
- Aethel-1x1.png → renombrar a `aethel.png`
- crono-velo-1x1.png → renombrar a `crono-velo.png`
- aethel-vertical.png → renombrar a `aethel_vertical.png`

Si no existe el asset, la app muestra un placeholder con la inicial del guía y su color de tema.
