# /doctor - Check Flutter Environment

Diagnose the Flutter development environment.

## Instructions

1. Run `flutter doctor -v` for detailed output
2. Parse and summarize:
   - Flutter version and channel
   - Dart version
   - Android toolchain status
   - Chrome/Web status
   - Windows toolchain status
   - Connected devices
3. Highlight any issues that need attention
4. Provide specific fix instructions for problems found
