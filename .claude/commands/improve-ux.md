# /improve-ux - UX/UI Improvement Agent

Deep analysis of user experience and interface design.

## Instructions

1. **Audit Current UI**
   - Read all files in `lib/screens/` and `lib/widgets/`
   - Map the complete user journey for task management
   - Identify friction points

2. **Analyze Key Flows**
   - Task creation: How many taps? Is it fast enough?
   - Task completion: Is it satisfying? Any celebration?
   - Task organization: Can users find things easily?
   - Navigation: Is it intuitive?

3. **Check UI Patterns**
   - Consistent spacing and typography
   - Clear visual hierarchy
   - Appropriate use of color
   - Loading and empty states
   - Error states and feedback

4. **Suggest Improvements**
   For each suggestion, provide:
   - Problem identified
   - Proposed solution
   - Implementation approach
   - Expected impact on user happiness

5. **Create Implementation Plan**
   Write specific code changes or new widgets needed.

## Output Format
```markdown
## UX Analysis: [Area]

### Current State
[Description of current experience]

### Issues Found
1. [Issue]: [Impact on user]

### Recommended Changes
1. [Change]:
   - Why: [Reason]
   - How: [Implementation]
   - Files: [Affected files]
```
